# Exercise 1.1 – Setting Up Python Environment

## Overview

This exercise covers setting up a Python development environment, creating virtual environments, installing packages, and running basic Python scripts.

## Steps Completed

1. Installed Python and Visual Studio Code.
2. Set up virtual environments (`cf-python-base`, `cf-python-copy`).
3. Installed packages: `bcrypt`, `ipython`.
4. Exported a `requirements.txt` file.
5. Created two simple Python scripts: `hello.py` and `add.py`.
